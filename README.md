# webtrain-tailwind
1. npm init -y
2. npx i tailwindcss
3. npx tailwindcss init  
4. npx tailwindcss -i ./src/styles.css -o ./public/output.css --watch (sesuaikan folder yang dituju)

## responsive design

- xl: 1280px
- 2xl: 1536px
- md: 768px
- sm: 640px
